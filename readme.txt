Geo_calc - это консольная программа, которая позволяет вычислять 
площади различных геометрических фигур: круга, прямоугольника и 
треугольника. 

Инструкция:

В командной строке (терминале) перейдите в директорию, в которой 
находится проект.

Для запуска программы Geo_calc, выполните следующую команду:

python Geo_calc.py
